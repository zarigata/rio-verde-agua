Sobre o Projeto

Trabalho de Conclusao de Curso

Este projeto constitui o componente pratico do Trabalho de Conclusao de Curso (TCC) na area de Gestao de Recursos Hidricos.

A plataforma foi desenvolvida como uma aplicacao web estatica, sem dependencias de servidor, utilizando exclusivamente tecnologias client-side: JavaScript (ES6+), Leaflet para mapas interativos e Chart.js para visualizacoes graficas.

Todos os dados utilizados sao provenientes de fontes publicas oficiais e possuem proveniencia registrada no codigo e na documentacao.

Tecnologias

  • JavaScript ES6+ (vanilla, sem frameworks)
  • Leaflet 1.9.4 — mapas interativos
  • Chart.js — visualizacoes graficas
  • GitHub Pages — hospedagem estatica
  • CSS Custom Properties — design system com dark mode

Arquitetura

  • Multi-pagina estatica (7 paginas HTML)
  • Modulos ES para logica separada por responsabilidade
  • Dados em JSON (sem backend)
  • Zero dependencias de build
